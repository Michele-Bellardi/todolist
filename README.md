# To Do List
## Descrizione del Progetto
Un'app ToDo sviluppata in Flutter con 2 schede: una che mostra le "task" da svolgere o già svolte e una che mostra le statistiche.

## Logica dell'applicazione
In questa applicazione To‑Do List l’utente può creare, visualizzare e gestire una lista di attività da svolgere. La schermata principale mostra l’elenco dei task inseriti, ognuno rappresentato da una voce selezionabile: premendo sulla casella di controllo accanto a ogni elemento è possibile marcarlo come completato o ripristinarlo come “da fare”.
Oltre alla gestione dei singoli task, l’applicazione offre la possibilità di creare più liste distinte (ad esempio “Scuola”, “Casa”, “Spesa”, ecc.). Attraverso il menu laterale è possibile selezionare la lista desiderata o crearne una nuova, ognuna con i propri elementi indipendenti dalle altre.
Nella parte superiore della pagina è presente un pulsante dedicato all’aggiunta di nuovi task: premendolo si apre una finestra di dialogo che permette all’utente di inserire il nome dell’attività da aggiungere alla lista.
La seconda schermata dell’app, accessibile tramite la barra di navigazione inferiore, è dedicata alle statistiche. In questa sezione vengono mostrati il numero totale di task presenti nella lista corrente, quanti sono stati completati, quanti restano da svolgere e la percentuale di efficienza calcolata in base ai progressi dell’utente.
Nel caso in cui siano presenti più liste, le statistiche si aggiornano automaticamente in base alla lista selezionata, permettendo così un monitoraggio chiaro e immediato dell’avanzamento per ciascun gruppo di attività.

## Per installare l'applicazione su Windows:
### Clonare la repository remota all'interno del computer:
da terminale, digitare: 
```bash
git clone https://github.com/Michele-Bellardi/todolist.git
```
### Avviare il progetto su Windows:
1. aprire il progetto con Android Studio o Visual Studio Code;
2. avviare un emulatore Android;
4. avviare il progetto.

## Per instalare l'applicazione su android:
Dopo aver aperto il progetto su Windows:
1. generare il file .apk da Android Studio;
2. trasferire il file generato all'interno della meoria del cellulare Android;
3. aprire il file .apk dal cellulare e seguire la procedura di installazione indicata;
4. aprire l'applicazione appena installata.
